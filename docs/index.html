🚀 Instalación en 3 Pasos

1

Clona el Proyecto

git clone https://github.com/erbolamm/plaud-assistant.git

Descarga el código fuente completo y listo para compilar.

2

Personaliza la Configuración

edita lib/config/app_config.dart

Añade nombres, familia, edad y configura tu API preferida.

3

Compila e Instala

flutter build apk --release

Genera el APK e instálalo en cualquier Android 6.0+.

� Conversación Natural y Segura

🤖 Asistente IA que habla como persona

Plaud Assistant mantiene conversaciones naturales adaptadas a cada niño/a, pero siempre queda claro que es un asistente digital:

  • 🎭 Respuestas emocionales: Detecta frustración y ofrece apoyo
  • 🧠 Contenido educativo: Adaptado según la edad (configurable)
  • 🌟 Datos curiosos: Aprendizaje divertido y espontáneo
  • 💬 Lenguaje natural: Como un amigo digital, no como una máquina
  • 🔒 100% seguro: Sin datos personales ni tracking

Ejemplo de conversación real:

Niño: "No entiendo esta matemática, es muy difícil 😤"
Plaud: "Uf, entiendo tu frustración 😤 A veces las matemáticas parecen un rompecabezas gigante. ¿Y si lo vemos como un juego? Eres una exploradora que busca el tesoro del número correcto. ¡Tú puedes! 🌟 ¿Qué parte te parece más complicada?"
