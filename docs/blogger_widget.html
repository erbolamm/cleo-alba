ClawMobil: Dale un cerebro a tu antiguo móvil

Reacondicionamos dispositivos antiguos convirtiéndolos en servidores IA personales. Sin cloud. Sin suscripciones.

Open Source · Libre · Tuyo
❤️ Apoya el proyecto: 💳 PayPal ☕ Ko-fi ⭐ GitHub

¿Tienes un smartphone cogiendo polvo en el cajón? Con ClawMobil y su wizard guiado puedes transformar ese hardware en un potente servidor de IA local. Este proyecto permite ejecutar modelos de lenguaje (LLM) y visión artificial directamente en el dispositivo, creando un cerebro digital de bolsillo sin depender de suscripciones externas.

⚠️ ADVERTENCIA CRÍTICA

NO USES TU TELÉFONO PERSONAL. Este proceso instala servicios que pueden comprometer la privacidad de tu terminal principal. Usa un dispositivo dedicado, realiza un factory reset y no vincules tu cuenta principal de Google.

¿Qué puede hacer tu nuevo cerebro?

🗣️
Chat por Voz Respuestas inteligentes con voz sintetizada directamente desde el terminal.
📸
Visión IA Identificación de objetos y lectura de texto mediante la cámara del smartphone.
✈️
100% Offline Uso de Ollama y whisper.cpp para funcionar en zonas sin cobertura.

🛠️ Cómo empezar

1
Descarga el repositorio desde GitHub y abre empezar.html
2
El wizard genera un prompt que pegas en tu asistente IA (Copilot, Cursor, Gemini...)
3
Tu IA configura todo automáticamente: Termux, SSH, Ollama y más

Stack Tecnológico

Frontend Flutter 3.24
Backend Python 3.12 + Node.js
IA Engine Groq (Cloud) / Ollama (Local)
Gateway OpenClaw (Telegram, SSH, Automación)

🚀 Instalación rápida

git clone https://github.com/erbolamm/ClawMobil.git
cd ClawMobil
# Abre empezar.html en tu navegador
