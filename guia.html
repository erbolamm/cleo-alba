Evaluación de tu dispositivo

Selecciona tus especificaciones arriba para ver qué puede hacer tu teléfono.

⚖️ Termux vs Root — Pros y riesgos

✅ Termux (sin root)

Seguro — no modifica el sistema
Reversible — desinstalar y listo
Fácil — instalar desde F-Droid
⚠️ Algo más lento por la capa proot

⚠️ Root

Más rápido — acceso directo al hardware
Más control — puedes quitar bloatware
⚠️ Puede dejar el teléfono inservible (brick)
⚠️ Anula la garantía (si queda alguna)

Instalación

Sigue estos pasos en orden. Los comandos se pueden copiar tocando el botón.

💡 ¿Tienes un IDE con IA? (VSCode, Cursor...)

Si tienes este proyecto abierto en tu editor con IA, puedes copiar el prompt del Panel de Despliegue y dejar que tu IA haga todo esto por ti automáticamente.

1

Instala Termux desde F-Droid

No uses la versión de Google Play — está desactualizada. Descarga desde F-Droid:

Enlace
https://f-droid.org/es/packages/com.termux/
2

Actualiza Termux y paquetes base

Abre Termux y ejecuta estos comandos uno por uno:

Termux
# Actualizar paquetes pkg update -y && pkg upgrade -y # Instalar lo esencial pkg install openssh nodejs-lts git -y # Iniciar SSH (para controlar desde PC) sshd
3

Configura SSH para control remoto

Esto te permite escribir los comandos desde tu PC en vez del teléfono:

Termux
# Pon una contraseña para SSH passwd # Tu IP del teléfono (anótala) ifconfig wlan0 | grep inet
Desde tu PC
# Conectar por SSH (cambia la IP) ssh -p 8022 IP_DEL_TELEFONO
4

Instala Debian (proot-distro)

Esto crea un Linux completo dentro de Termux. Necesario para OpenClaw:

Termux
# Instalar proot-distro pkg install proot-distro -y # Instalar Debian proot-distro install debian # Entrar en Debian proot-distro login debian
5

Instala OpenClaw

Dentro de Debian, instala OpenClaw:

Debian (dentro de Termux)
# Instalar dependencias apt update && apt install -y curl # Instalar OpenClaw curl -fsSL https://get.openclaw.dev | bash # Configurar (sigue el wizard) openclaw wizard
5

Verificación

Comprueba que todo funciona correctamente.

✓

✓
