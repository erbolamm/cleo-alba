Preparación

Prepara el dispositivo antes de empezar

1

Restablecimiento de fábrica

Ve a Ajustes → Sistema → Restablecer. Esto borra todo. Asegúrate de no tener nada importante.

2

Configuración inicial mínima

Salta la cuenta de Google o crea una nueva solo para esto. Conecta solo al WiFi. No metas tu SIM personal.

3

Activa la Depuración USB

Ve a Ajustes → Acerca del teléfono, toca 7 veces en "Número de compilación". Luego activa Depuración USB en Opciones de desarrollador.

4

Instala Termux desde F-Droid

Descarga F-Droid (f-droid.org) e instala Termux y Termux:Boot desde ahí. No uses la versión de Google Play.

5

Conecta el dispositivo al Mac/PC por USB

Autoriza la conexión ADB cuando aparezca el diálogo en el teléfono. Verifica con adb devices.

Consejos y Trucos Maestros

🔄 Rotación de claves API (Anti-Bloqueo)

Si usas mucho las APIs de Groq o Gemini, es probable que alcances los límites gratuitos. Configura un sistema de rotación rápida para no detener tu flujo de trabajo:

1. Edita ~/.bashrc y añade todas tus claves con nombres descriptivos:

export GROQ_API_KEY="clave_principal" export GROQ_KEY2="clave2" export GROQ_KEY3="clave3" export GEMINI_API_KEY="clave_principal_gemini" export GEMINI_KEY2="clave2_gemini"

2. Para rotar cuando una se agote (sin editar el archivo cada vez):

export GROQ_API_KEY="$GROQ_KEY2" source ~/.bashrc

3. Reinicia el gateway para aplicar el cambio inmediatamente:

pkill -f openclaw-gateway nohup openclaw gateway run --allow-unconfigured --force > /tmp/openclaw-gateway.log 2>&1 &

¿Listo? ¿Arrancamos el gateway ahora con la nueva clave?
