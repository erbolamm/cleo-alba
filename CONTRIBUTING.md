# Contributing to ClawMobil 🚀

First off, thanks for taking the time to contribute! This project is born from the desire to give a second life to high-end vintage mobile devices (like the Huawei P10) as AI-powered desktop companions.

## How to Contribute

### 1. Reporting Bugs / Feats
If you find a bug or achieve a "feat" (a cool configuration or new skill), use the **"Report Bug/Feat"** button in the app or the Mission Control panel. 
It will generate a `.md` report that you can send to `info@apliarte.com`.

### 2. Developing Skills
New skills should be added to the `skills/` directory. Each skill must have its own documentation and helper scripts.

### 3. Flutter App Improvements
- We use native Flutter widgets for the avatar (NeonFaceWidget).
- Avoid adding large dependencies like game engines.
- Optimize for battery and CPU usage.

## Code of Conduct
Be kind, be helpful, and help us turn old phones into powerful AI tools!
