Flujo guiado (desde cero)

Tarda 5-10 minutos para dejar un clon funcional de ClawMobil en tu equipo.

1. Abre la carpeta en VS Code

Menú `Archivo > Abrir carpeta...` y selecciona el repositorio `ClawMobil`.

2. Lanza esta guía local

Puedes abrir `index.html` con doble clic o desde VS Code (extensión Live Server opcional).

3. Entra al asistente

Haz clic en Abrir asistente, elige dispositivo, funciones, idioma y permisos.

4. Genera y aplica tu perfil

El asistente te entrega `config_profile.json`, prompt modular y comando shell para activar tu instancia.

5. Valida y comparte mejoras

Ejecuta scripts de evaluación/discovery y promueve solo cambios reutilizables.

bash scripts/local_config_select.sh --json bash scripts/local_config_evaluate.sh bash scripts/local_config_discovery.sh

Rutas que sí debes tocar

`Mis_configuraciones_locales/dispositivos/<tu_dispositivo>/` Tu zona local de pruebas y personalización
`empezar.html` Wizard para generar perfil + prompt + comando
`docs/LOCAL_CONFIG_FRAMEWORK.md` Cómo evaluar, promover y revertir cambios
Mantén secretos y experimentos en `Mis_configuraciones_locales`. Así cada fork es independiente y reversible.
